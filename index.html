Welcome to CodaLab.

CodaLab is an open source, web application platform with Azure integration for experimentation and competition. CodaLab is written using Python, Django, and Foundation and can be installed easily to provide a space for users to collaborate on experiments and compete in challenges. CodaLabs Azure integratoin provides the ability to leverage and share cloud-based computation and data resources among users of a CodaLab instance.

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Getting Started

There are two ways to use CodaLab: host a CodaLab instance, or use an existing CodaLab instance. Additionally, since CodaLab is open-source, we invite developers to contribute enhancements.

Checkout the CodaLab wiki for our current documentation.
